
# 타임트리 컨설팅 랜딩 페이지

13년 경력의 중학교 과학 교사 출신 컨설턴트 "정 선생님"을 위한 전문 입시 컨설팅 서비스 "타임트리"의 공식 랜딩 페이지입니다.

## 프로젝트 개요

선생님의 전문성과 경험을 강조하여 학부모와 학생들에게 신뢰를 주고, 체계적인 데이터 기반 컨설팅의 강점을 부각시키는 것을 목표로 제작되었습니다.

## 주요 기능

- **히어로 섹션**: 서비스의 핵심 가치를 전달하는 인상적인 첫 화면
- **핵심 강점**: 웹 컴포넌트를 활용한 깔끔한 카드 UI로 서비스의 세 가지 핵심 강점 소개
- **창업 및 실행 계획**: 정부 지원 사업과 실행 타임라인을 안내하여 전문성과 신뢰도 부여
- **연락처 정보**: 전화 연결이 가능한 연락처를 포함하여 즉각적인 상담 유도
- **반응형 디자인**: 모바일, 태블릿, 데스크탑 등 모든 기기에서 최적화된 화면 제공

## 기술 스택

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript (ES6+)
- Web Components
