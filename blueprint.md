
# Blueprint: 타임트리 컨설팅 랜딩 페이지

## 1. 개요

13년 경력의 중학교 과학 교사 출신 컨설턴트 "정 선생님"을 위한 전문 입시 컨설팅 서비스 "타임트리"의 공식 랜딩 페이지입니다. 선생님의 전문성과 경험을 강조하여 학부모와 학생들에게 신뢰를 주고, 체계적인 데이터 기반 컨설팅의 강점을 부각시키는 것을 목표로 합니다.

## 2. 디자인 및 기능 명세

### 가. 디자인 컨셉
- **전체 분위기**: `background.jpg` 이미지를 활용하여 차분하고 신뢰감 있는 배경을 조성합니다. 텍스트 가독성을 위해 반투명 오버레이를 추가합니다.
- **아이콘**: `icon.jpg`를 페이지의 파비콘(Favicon) 및 로고 영역에 활용하여 브랜드 정체성을 나타냅니다.
- **색상 팔레트**: 신뢰를 상징하는 네이비와 블루 계열을 주 색상으로 사용하고, 포인트 색상으로 골드 또는 밝은 계열의 색상을 사용하여 전문성을 강조합니다.
- **타이포그래피**: 가독성 높은 `Noto Sans KR`과 같은 고딕 계열 폰트를 사용하여 내용을 명확하게 전달하고, 제목과 본문의 크기, 굵기 차이를 두어 정보의 위계를 표현합니다.
- **레이아웃**: 섹션별로 정보를 명확하게 구분하고, 카드 디자인, 아이콘 등을 활용하여 시각적 이해도를 높입니다. 모바일 화면에서도 최적화된 반응형 디자인을 적용합니다.

### 나. 페이지 구성
1.  **헤더 (Header)**
    *   로고: `타임트리` 텍스트 로고
    *   네비게이션: `타임트리 소개`, `핵심 강점`, `정부 지원`, `실행 계획`, `상담 문의` 등 각 섹션으로 이동하는 링크
2.  **히어로 섹션 (Hero Section)**
    *   핵심 슬로건: "13년 차 과학 교사의 데이터 기반 입시 전략, 타임트리"
    *   배경: `background.jpg` 이미지 사용
    *   CTA 버튼: "지금 바로 상담 신청하기"
3.  **타임트리 소개 (About Section)**
    *   정 선생님의 전문성과 경력을 소개하는 단락.
    *   `13년 차 현직 교사 출신`, `과학적/체계적 관리`, `중등-고등 연계 이해` 등 핵심 키워드 강조.
4.  **핵심 강점 (Strengths Section)**
    *   `압도적인 신뢰도`, `체계적인 관리 능력`, `중등-고등 연계의 이해`를 각각의 카드로 구성하여 시각적으로 표현합니다.
5.  **2026년 창업 지원 프로그램 (Government Support Section)**
    *   `예비창업패키지`, `생애최초 청년창업지원사업`, `여성 특화 지원` 등 제공된 정보를 아코디언 메뉴나 탭으로 구성하여 깔끔하게 제공합니다.
6.  **2026년 상반기 실행 타임라인 (Timeline Section)**
    *   1월부터 4월까지의 계획을 시각적인 타임라인 형태로 표현합니다.
7.  **현실적인 조언 (Advice Section)**
    *   `CEO 마인드`, `데이터 확보`, `겸직 금지 확인` 등 중요한 조언을 강조하여 전달합니다.
8.  **상담 문의 및 CTA (Contact Section)**
    *   "정 선생님"의 이름과 연락처(`010-2063-9627`)를 명시합니다.
    *   전화 아이콘과 함께 클릭 시 바로 전화 연결이 가능한 `tel:` 링크를 제공합니다.
    *   간단한 문의 폼(선택 사항)을 추가할 수 있습니다.
9.  **푸터 (Footer)**
    *   `© 2024 TimeTree. All rights reserved.` 와 같은 저작권 정보를 표기합니다.

### 다. 인터랙션
- **Web Components**: 정보 카드, 타임라인 단계 등 반복되는 UI 요소를 웹 컴포넌트(`feature-card`, `timeline-step`)로 제작하여 코드의 재사용성과 유지보수성을 높입니다.
- **애니메이션**: 스크롤 시 각 섹션의 콘텐츠가 부드럽게 나타나는 효과를 추가하여 동적인 느낌을 줍니다.
- **반응형**: 모바일, 태블릿, 데스크탑 등 모든 화면 크기에서 콘텐츠가 잘 보이도록 미디어 쿼리를 사용합니다.

## 3. 현재 진행 계획

1.  **`blueprint.md` 파일 작성 (완료)**
2.  **`style.css`**: 기본 스타일, 배경, 색상, 폰트 등 CSS 초기 설정
3.  **`index.html`**: 제공된 텍스트를 기반으로 HTML 구조 설계 및 콘텐츠 배치
4.  **`main.js`**: 웹 컴포넌트(카드) 정의 및 기본 인터랙션 스크립트 작성
5.  **검토 및 수정**: 전체적인 디자인과 기능 검토 후 세부 수정 진행
